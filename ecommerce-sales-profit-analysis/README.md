# E-Commerce Sales & Profit Analysis

## Project Overview

This project presents an exploratory data analysis (EDA) of an e-commerce dataset to evaluate sales performance, profitability patterns, and customer segment behavior. The objective is to derive meaningful business insights using structured data analysis and visualization techniques.

---

## Objectives

* Perform descriptive statistical analysis to understand data distribution
* Identify and handle data quality issues
* Analyze monthly sales trends and determine highest and lowest performing months
* Evaluate sales performance across product categories and sub-categories
* Analyze monthly profit trends and identify peak profit periods
* Examine profitability at category and sub-category levels
* Compare sales and profit across customer segments
* Compute and evaluate sales-to-profit ratio
* Analyze yearly sales and profit trends

---

## Data Preparation

* Checked descriptive statistics using summary functions
* Inspected data types and null values
* Converted date columns to datetime format
* Created derived features including Order Year and Order Month
